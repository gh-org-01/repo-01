SQL Query Scripting Hook

Support for SQL queries in Liferay script console

Download this plugin as a .war file

What's this plugin?

With this plugin, you will be able to make SQL queries in Liferay console script and to see the results in pretty formated HTML or in CSV.

See it in action

Usage details

You can @mention a GitHub username to generate a link to their profile. The resulting <a> element will link to the contributor's GitHub Profile. For example: In 2007, Chris Wanstrath (@defunkt), PJ Hyett (@pjhyett), and Tom Preston-Werner (@mojombo) founded GitHub.
